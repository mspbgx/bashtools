# bashtools
helpful bash scripts

## Installation
1. Checkout: 	      
```
git checkout https://github.com/mspbgx/bashtools.git
```
2. Move to ~/.bin:    
```
mv bashtools ~/.bin 
```
3. Make scripts executable:
```
cd ~/.bin/tools && chmod +x *
```
4. Extend ~/.bashrc:  
```
if [ -f ~/.bin/.bashrc ]; then
  . ~/.bin/.bashrc
fi
```
5. Reload bash:	      
```
source ~/.bashrc
```
