# bashtools
helpful bash scripts

## Installation
1. Checkout: 	      git checkout https://github.com/mspbgx/bashtools.git
2. Move to ~/.bin:    mv bashtools ~/.bin 
3. Extend ~/.bashrc:  if [ -f ~/.bin/.bashrc ]; then
        	        . ~/.bin/.bashrc
		      fi
4. Reload bash:	      source ~/.bashrc
